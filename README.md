# 📚 PlusWork - Smart Assignment Submission System

## Features
- ✅ Teachers create assignments with deadlines
- ✅ Students upload submissions (PDF, DOCX, ZIP)
- ✅ Automatic late submission detection
- ✅ Bulk download all submissions as ZIP
- ✅ File validation (type & size)
- ✅ Resubmission limit (max 3 attempts)
- ✅ Grade & feedback system

## Installation
1. Install XAMPP
2. Copy folder to `C:\xampp\htdocs\assignment_system`
3. Start Apache & MySQL in XAMPP
4. Import `database.sql` in phpMyAdmin
5. Visit `http://localhost/assignment_system`
